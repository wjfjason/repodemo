# repodemo

一个简单的 Python 示例项目，用于演示基本的 Python 脚本功能和经典算法实现。

## 功能

- 打印 "hello world" 和当前工作目录
- 冒泡排序算法实现（含提前退出优化）

## 项目结构

```
/workspace/
├── .gitignore    # Git 忽略文件配置
├── README.md     # 项目说明文档
└── a1.py         # 主程序文件
```

## 运行方式

确保已安装 Python 3.x，然后在项目根目录下执行：

```bash
python a1.py
```

## 预期输出

```
hello world
/workspace
排序前: [64, 34, 25, 12, 22, 11, 90]
排序后: [11, 12, 22, 25, 34, 64, 90]
```
