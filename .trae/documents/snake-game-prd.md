# 贪吃蛇游戏 - 产品需求文档 (PRD)

## 1. 产品概述

一款具有复古街机风格的网页版贪吃蛇游戏。玩家控制蛇在有限空间内移动，吃掉食物增长身体，撞墙或撞自身即游戏结束。游戏追求流畅的操作手感、复古霓虹视觉效果和令人上瘾的游戏体验。

**目标用户**：休闲游戏玩家、复古游戏爱好者
**核心价值**：简单上手、快速开始的即时娱乐体验

## 2. 核心功能

### 2.1 游戏核心机制
- **蛇的移动**：四方向移动（上下左右），蛇身连续移动
- **食物系统**：随机生成食物，吃掉后蛇身增长一节
- **碰撞检测**：撞墙壁或自身身体时游戏结束
- **计分系统**：每吃一个食物得10分，实时显示当前分数和历史最高分

### 2.2 游戏控制
- **键盘控制**：方向键或WASD控制蛇的移动方向
- **开始游戏**：按空格键或点击开始按钮
- **暂停/继续**：按P键或点击暂停按钮
- **重新开始**：游戏结束后按空格键或点击重新开始按钮

### 2.3 游戏状态
- **开始界面**：显示游戏标题、最高分、开始按钮
- **游戏中**：显示游戏画面、当前分数
- **暂停状态**：显示"PAUSED"提示
- **游戏结束**：显示最终得分、最高分纪录、游戏结束提示、重新开始按钮

## 3. 核心流程

```
[开始界面] → 按空格/点击开始
     ↓
[游戏中] → 控制蛇移动
     ↓
吃食物？ → 是 → 分数+10，蛇身+1
     ↓ 否
撞墙/撞自身？ → 是 → [游戏结束]
     ↓ 否
继续移动
```

## 4. 用户界面设计

### 4.1 设计风格：霓虹复古街机风
- **主色调**：深色背景 (#0a0a0a) 配霓虹绿色 (#39ff14) 和粉红色 (#ff1493)
- **视觉效果**：
  - 霓虹发光效果（text-shadow, box-shadow）
  - 扫描线纹理覆盖
  - 像素化网格游戏区域
  - 复古CRT显示器边框效果
- **字体**：使用等宽像素风格字体

### 4.2 游戏画面布局
```
┌─────────────────────────────────┐
│         🐍 SNAKE GAME          │  ← 霓虹标题
│      当前分数: 0  最高分: 0     │
├─────────────────────────────────┤
│ ┌─────────────────────────────┐ │
│ │                             │ │
│ │      游戏画布 (Canvas)       │ │  ← 400x400像素
│ │                             │ │
│ └─────────────────────────────┘ │
├─────────────────────────────────┤
│   [空格]开始  [P]暂停  [方向键]移动 │  ← 控制提示
└─────────────────────────────────┘
```

### 4.3 游戏区域规格
- **画布大小**：400x400像素
- **网格划分**：20x20格（每格20像素）
- **蛇身**：绿色矩形，带发光效果
- **食物**：红色圆形，脉冲动画效果
- **背景**：深色网格线

### 4.4 响应式设计
- 桌面优先设计
- 移动端支持触摸滑动控制
- 最小宽度：450px

## 5. 技术实现

- **技术栈**：HTML5 + CSS3 + Vanilla JavaScript
- **渲染方式**：HTML5 Canvas
- **动画**：requestAnimationFrame
- **数据持久化**：localStorage存储最高分
- **无外部依赖**：纯原生实现
